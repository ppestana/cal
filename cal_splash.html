01 Project Overview

CAL is a data analytics system that quantifies deviations between expected and observed refereeing decisions in the Portuguese Primeira Liga (Liga Portugal Betclic), covering seasons 2017/18 to 2025/26.

For each referee, season, and decision type (yellow cards, red cards, fouls), the system calculates binomial Z-scores and a composite SuspicionScore that enables ranking and comparison across referees. The system covers 2,673 matches across 9 seasons with 98.4% referee coverage.

Important: CAL does not constitute accusation. It identifies statistically anomalous patterns that merit public scrutiny. Interpretation of results is always the responsibility of the user.

04 Technical Implementation

Modular pipeline in six independent phases, all running inside Docker containers on a Hetzner Cloud VPS (Ubuntu 24.04). The architecture separates data ingestion, feature engineering, statistical modelling, bias calculation, and visualisation into discrete, independently runnable modules.

Data sources: football-data.co.uk (match statistics — goals, cards, fouls) and Sofascore API (referee assignments). Both free of charge.

Statistical models: xYC expected yellow card model (Brier score 0.067), xRC expected red card model (Brier 0.113), xF expected fouls Poisson regression (MAE 3.3 fouls). Leave-One-Season-Out cross-validation. Benjamini–Hochberg FDR correction at 5% applied to all significance tests.

Bias engine: Z-scores per referee × season × decision type, temporal consistency analysis, peer comparison, referee × team interaction, three alert levels (monitor |Z|>1.0, suspect |Z|>2.5, anomaly |Z|>3.0).
